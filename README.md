# RCDE Frontend SDK

RCDEの機能をフロントエンドアプリケーションで利用するためのSDKです。

## インストール方法

npm、またはyarnを使用してインストールします。

```bash
npm install @i-con/frontend-sdk
# or 
yarn add @i-con/frontend-sdk
```

## 事前準備

RCDEのサイトでアプリケーションを作成します。
以下の手順に従ってください。
